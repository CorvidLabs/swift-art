# Changelog

## [0.1.0] - 2024-12-01

### Added
- Core types (Point2D, Point3D, colors, random)
- Noise generators (Perlin, Simplex, Worley, Fractal)
- Fractals (Mandelbrot, Julia, Sierpinski, Koch)
- L-Systems with Turtle graphics
- Particle systems with forces
- Cellular automata (Game of Life, Elementary CA)
- Geometry (polygons, Voronoi, tessellation)
- Color utilities (harmonies, gradients, palettes)
- ArtUI - SwiftUI integration
- ArtTerminal - ASCII art rendering
